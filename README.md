# 交互式流程图编辑器项目说明

## 网站简介
本项目是一个基于Web的交互式流程图编辑器，允许用户通过直观的图形界面创建、编辑和管理流程图。该编辑器旨在提高流程设计和管理的效率，适用于多种业务场景，如工作流程设计、业务流程重构等。

## 页面与功能
本网站包含以下6个主要页面及其功能：
1. **首页**：介绍编辑器的主要功能和特点，提供快速入门指南。
2. **编辑器页面**：核心功能页面，提供流程图的创建、编辑、保存和导出功能。
3. **模板库**：提供预设的流程图模板，帮助用户快速开始流程设计。
4. **帮助文档**：详细的使用指南和常见问题解答，辅助用户解决问题。
5. **示例展示**：展示使用编辑器创建的流程图示例，激发用户灵感。
6. **关于我们**：项目背景、开发团队介绍及联系方式。

## 多 AI 协作与验收
本项目采用多AI协作的方式完成，从需求分析到最终实现，多个AI模型分工合作：
- **需求分析AI**：负责理解项目需求，分解任务。
- **设计AI**：根据需求分析结果，进行页面设计和用户体验优化。
- **开发AI**：实现编辑器的核心功能和页面开发。
- **测试AI**：进行功能测试、兼容性测试和性能测试，确保项目质量。

项目通过以下步骤进行验收：
1. 功能验收：检查所有功能是否按预期工作。

3. 用户体验验收：测试网站的易用性和交互体验。

## GitHub Pages 部署与访问
本项目已部署到GitHub Pages，访问入口为`index.html`。用户可通过以下步骤访问：
1. 打开GitHub仓库页面。
2. 点击`Settings` > `GitHub Pages`，找到发布的网站链接。
3. 在浏览器中输入该链接，即可访问`index.html`并开始使用交互式流程图编辑器。
